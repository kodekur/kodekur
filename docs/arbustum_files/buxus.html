Самшит / Buxus

Ценится за густую красивую крону, блестящую листву и способность хорошо переносить стрижку. Используется для создания бордюров и групп. В одиночной посадке может украсить газон. В средней полосе на зиму рекомендуется укрывать лутрасилом для снижения вероятности зимнего иссушения. Выигрышно смотрится на фоне хвойных.
