Гортензия метельчатая Сандей Фрэйз / Hydrangea paniculata Sandae Fraise

Более компактная версия Vanille Fraise высотой до 1 м. Великолепно подходит для небольших садов. Красно-бордовые жёсткие побеги надёжно держат многочисленные крупные шапки соцветий, состоящих из стерильных цветков. При распускании цветки белые, затем быстро меняют цвет на насыщенно-розовый с вишнёвым оттенком, создавая эффект двуцветности. Цветёт с июля по сентябрь.
