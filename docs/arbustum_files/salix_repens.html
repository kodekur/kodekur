Ива ползучая Аргентеа / Salix repens Argentea

Низкий густой кустарник с распростёртыми побегами 0,5 м высотой и 1,0-1,5 м шириной. Листья мелкие, серебристо-голубые, сильноопушённые. Круглые золотистые соцветия-серёжки появляются до распускания листьев. Растёт медленно. Возможно формирование в штамбовой форме для получения эффекта плакучего деревца. Подходит для размещения на каменистых горках и на переднем плане смешанных посадок.

Размножение - по запросу.
