Лапчатка (пятилисточник) / Potentilla

Выносливый, сильноветвящийся кустарник с плотной полушаровидной кроной, достигающий высоты 0,5-1,5 м. Листья состоят из пяти листочков. Цветки 1,5-4,0 см в диаметре, чашевидные, одиночные, "разбросаны" по поверхности куста. Цветки быстро увядают, но постоянно заменяются вновь распускающимися. Цветение длится с мая по сентябрь.
Используется в качестве свободно растущей живой изгороди. Замечательно смотрится в полутени под кронами деревьев. Хорошо сочетается с барбарисами, ивами, спиреями, видовыми кустарниками. Низкорослые сорта рекомендуется размещать в каменистых садиках.
