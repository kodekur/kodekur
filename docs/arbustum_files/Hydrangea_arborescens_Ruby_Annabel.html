Гортензия древовидная Руби Анабель (Инвинсибелль Руби ) /
Hydrangea arborescens Ruby Annabel (Invincibelle Ruby )

Новинка! Высота 60-90 см. Бутоны тёмно-красные, внешняя сторона лепестков красно-розового цвета, внутренняя – сначала с бордовым оттенком, со временем переходит в розово-серебристый. Цветёт на побегах текущего года с июня до заморозков. Листья тёмно-зелёные. Побеги крепкие. Обладатель награды: Бронзовая медаль, Плантариум 2016.
