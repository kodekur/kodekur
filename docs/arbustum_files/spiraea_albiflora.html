Спирея японская Альбифлора / Spiraea japonica Albiflora

Единственный белоцветковый сорт спиреи японской. Низкий кустарник с приподнятыми ветвями высотой до 0,8 м. Листья эллиптические, светло-зелёные. Белые цветки собраны в плоские зонтики. Цветёт с июля по сентябрь.
