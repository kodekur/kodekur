Ожика лесная / Luzula sylvatica

Высота до 50 см. Листья ярко-зелёные, плоские, кожистые, блестящие, по краю реснитчатые. Соцветие сложное, метельчато-развесистое, каштанового цвета. Цветение в мае – июне. Листья зимующие.
