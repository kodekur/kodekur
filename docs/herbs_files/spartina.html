Спартина / Spartina

Спартина перистая Ауреомаргината / Spartina pectinata Aureomarginata

Экзотичный крупный куст до 1,5 м высотой. Листья глянцевые, дуговидно-изогнутые с яркими жёлтыми полосами по краям. Необычные, жёсткие, щётковидные соцветия образуются на вершинах двухметровых побегов в конце лета. Прекрасно выступает в роли акцента любой садовой композиции.
