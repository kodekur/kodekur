Калимерис надрезаннолистный Блю Стар /
Kalimeris incisa Blue Star

Неприхотливае растение. Образует немного раскидистый ажурный куст высотой до 120 см. Листья тёмно-зелёные ланцетовидные. Цветёт некрупными ромашковидными светло-голубыми цветками с ярко-жёлтой серединкой. Цветёт продолжительно с июля по сентябрь.
Используется в миксбордерах, группах, цветниках природного стиля. Благодаря высоте можно высаживать между кустарниками.
Партнеры: многолетние астры, вероникаструмы, высокие горцы, клопогоны, высокие кровохлёбки.
