Пион ИТО-гибрид Кэллис Мемори /
Paeonia Iton hybrids Callie‘s Memory

Anderson, 1990 г. Высота 90 см. Полумахровый с открытым центром. Цветок крупный диаметром 20 см, от золотисто-кремово-жёлтого до персикового с каштановыми пятнами у основания центральных лепестков. Есть тычинки и пестики. На стебле 2-3 бутона, которые раскрываются постепенно. Ароматный. Продолжительное цветение в средние сроки.

На размножении.
