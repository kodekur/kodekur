Телиптерис (фегоптерис) / Thelypteris (phegopteris)

Телиптерис буковый (фегоптерис связывающий) / Thelypteris phegopteris (phegopteris connectilis)

Невысокое растение до 30-40 см высотой с причудливо-орнаментальными, желтовато-зелёными треугольными дважды перистыми листьями на длинных тонких черешках. Стабильно декоративно с весны до заморозков. Рекомендуется для создания напочвенного покрова на затенённых участках в соседстве с астильбами, бузульниками, декоративными травами, купеной, медуницами, роджерсией, хостами.
