Анемона хубейская Прикос / Anemone hupehensis Praecos

Высокорослое растение высотой 0,5-0,9 м с мощными цветоносами. Листья серовато-зелёные, опушённые, крупные, трижды перисто-рассечённые с зубчатыми лопастями. Соцветия многоярусные, состоящие из множества нежно розовых цветков диаметром до 6 см. Цветёт с августа по октябрь. Используется в смешанных посадках с осеннецветущими многолетними растениями: астрами, флоксами метельчатыми.

На размножении.
