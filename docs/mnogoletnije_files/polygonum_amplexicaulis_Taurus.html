Горец стеблеобъемлющий ( свечевидный) Taurus /
Persicaria amplexicaulis Taurus

Высота 60-80 см. Листья тёмно-зелёные, глянцевые. Нижние листья на длинных черешках, выше они становятся стеблеобъемлющими. Соцветия длинные свечевидные, тёмно-красного цвета. Цветёт в июле-августе. Отлично подходит для срезки. Не агрессивен! Используется в садах природного облика совместно с амсонией, астранцией, колокольчиками, василистниками, вербейниками, верониками , нивяниками.

На размножении.
