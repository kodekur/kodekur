Увулярия / Uvularia

Увулярия крупноцветковая / Uvularia grandiflora

Редкое, элегантное растение с многочисленными стеблями высотой 40-50 см, на концах которых расположены по 2-3 поникающих цветка ярко-жёлтого цвета колокольчатой формы длиной около 4 см. Листья заострённо-элиптические, стеблеобъемлющие, слегка пушистые с внутренней стороны. Цветёт мае-июне. Естественно смотрится среди тенелюбивых культур: астильб, бузульников, ожики, клёнолистника, медуниц, папоротников, хост.
