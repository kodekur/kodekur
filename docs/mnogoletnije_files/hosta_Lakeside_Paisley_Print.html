Хоста Lakeside Paisley Print / Hosta Lakeside Paisley Print

Chastain, 2006 г. Хоста группы M. Листья сердцевидные, плотные, гофрированные, центр кремово-белый, кайма широкая тёмно-зелёная, сильно перьевидная, ближе к центру листа перья цвета шартрез, длинные иногда доходят до середины листа. Цветки лавандовые. Цветёт в июле.
Американской ассоциацией производителей хост присвоен титул «Хоста года» 2019.
