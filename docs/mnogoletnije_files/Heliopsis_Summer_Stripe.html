Гелиопсис подсолнечниковидный Самма Страйп /
Heliopsis helianthoides Summer Stripe

Растение высотой 60-70 см с прямостоячими тёмно-красными стеблями. Очень нарядные светло-зелёные листья с ярко-выраженными тёмно-зелёными прожилками. Соцветия–корзинки оранжево-жёлтые диаметром около 8 см, черезвычайно долговечны.
