Сангвинария / Sanguinaria

Многолетнее весеннецветущее растение. Нечасто встречается в культуре. Как высокодекоративное оригинальное растение рекомендуется использовать для посадки под кронами крупных листопадных деревьев, для создания цветущего пятна в парадных тенистых цветниках
