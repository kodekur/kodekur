Пион гибридный Ред Чарм / Paeonia hybrids Red Charm

Гибрид P. albiflora x P. Officinalis. Один из лучших красных гибридов. Чемпион многих выставок. Цветок махровый, плотный, красивой бомбовидной формы, насыщенного тёмно-малиново-красного цвета. Диаметр цветка 20 см. Ранний.

"Чемпион выставки" (Grand Champion) 1951, 1965, "Золотая медаль" (Gold Medal) 1956.
