Дицентра / Dicentra

Неприхотливое долгоцветущее растение. Дугообразные поникающие односторонние кисти характерных цветков, похожих по форме на маленькие сердечки, из которых капают слезинки.
Превосходно смотрится одиночным кустом. Цветёт в мае-июне. Используется в сочетании с аквилегиями, колокольчиками, купеной, нарциссами, печёночницами, тюльпанами.
