Пенстемон наперстянковый Хаскер Ред /
Penstemon digitslis Husker Red

Один из самых морозостойких видов. Стебли прочные, слабоветвистые высотой 60-100 см. Листва бронзово-красная. Прикорневые листья зимующие. Во время цветения образует массу трубчатых воздушных розоватых цветков. Цветёт долго - с июня по август. Высаживают группами в сборные цветники и обсаживают низкими бордюрными растениями: арабисами, барвинками, камнеломоками, молодилами, седумами, пахистимой, прострелами , тимьянами, флоксами шиловидными, ясколкой. Хорошо растёт в уличных вазах. Соцветия пригодны для срезки.
